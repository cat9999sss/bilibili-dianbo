### 基于弹幕姬sdk开发的Bilibili点播助手
#### 弹幕姬插件
弹幕姬安装参见下面的网站
https://www.danmuji.org/

#### 功能
获取用户弹幕：点播****
**** = 0000 - 9999
播放对应的视频

#### 使用方法
下载插件放到弹幕姬插件目录中，重启弹幕姬
在创建文件夹e:/video
在video中放入视频文件，目前只支持****.mp4
